總共有3總鏈 , 都是防止資料串改
- 私有鏈
    1. 資料公開對象都是經過驗證過的
    2. 不須多個驗證節點 , 交易速度較快
    3. 發生異常 也較快發現來源 
- 公有鏈
    1. 去中心化
    2. 交易速度比較緩慢
    3. 資料公開
    
- 聯盟鏈
    1. 只有部分的去中心化 , 每個機構都有自己的中心化
    2. 指定共識者(節點)
    3. 聯盟鏈其實是私有鏈的一種
    4. 每次的活動權限都可定義
